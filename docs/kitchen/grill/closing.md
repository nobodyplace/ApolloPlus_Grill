# 明日の仕込みを書く

チェックすべき仕込みは次の通り。必要な分量は料理長に確認しながら決めてください。

- 餃子
- 大根おろし
- ねぎ
- 上賀茂野菜サラダ用ドレッシング
- 玉ねぎドレッシング
- デミソース
- タルタルソース
- クレームブリュレ
- バニラアイス
- ガーリックチップ
- 漬け物
    - 漬け物を漬けた日は「つけもの」つけなかったひは「ぬかまぜ」と書く
- ヒレ衣
    - 2日に1回
- ボイル野菜
- ちぎり野菜
